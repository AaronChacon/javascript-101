# `console.log()` - 🖨 Mostrar información - 🌎 Global
## Ejemplo ```javascript $ console.log(1989) $ console.log(13, 10, 1989) ```
# Tipos de Dato - 🔢 Numeros - `Number` - 🔗 Cadenas de Caracteres - `String` - 0️⃣1️⃣ Booleanos - `Boolean`
## `Number` - Enteros - Decimales - Positivos - Negativos - Operaciones Aritméticas ```javascript 1 -1 9.5 0 99999 ```
### `String` - Conjuntos de caracteres - Comillas (simples o dobles) ```javascript 'Esto es un String' "Y esto también" ```
## `Boolean` - Lógica binaria (`0` o `1`) - `true` o `false` ```javascript true false ```
## Ejemplos ```javascript console.log(1, 2, 3) console.log('Hola JavaScript 👋') console.log(true, false) ```
# 📦 Variables - Referencias a valores 🤔 - Reutilización - `var`
## Ejemplo ```javascript var miVariable = 'Esto es una variable' console.log(miVariable) ```
# 😱 `undefined` Tipo de dato ```javascript var miVariableSinValor console.log(miVariableSinValor) ```
# 📱 Operadores - Asignación - Aritméticos - Comparación
## Asignación ```javascript var miVariable = true var miOtraVariable = 25 ```
## Aritméticos ```javascript var suma = 1 + 9 // 10 var resta = 3 - 2 // 1 var division = 100 / 10 // 10 var multiplicación = 10 * 10 // 100 var x = 1 var y = 2 var z = x + y // 3 ```
## Aritméticos ```javascript var x = 1 var y = 2 var z = x + y // 3 ```
## Comparación ```javascript var igual = 1 == 1 // true var distinto = 2 != 1 // true var mayor = 3 > 2 // true var menor = 3 < 2 // true ```
# 🔀 Condicionales - Toma de decisiones - Evaluar condiciones - `if` - `else`
## `if` ```javascript var haceFrio = true if (haceFrio) { console.log('❄️') } ```
## `else` ```javascript var haceFrio = false if (haceFrio) { console.log('❄️') } else { console.log('☀️') } ```
# 🗂 `Array` - Colecciones de datos - Indices
## Ejemplo ```javascript var numeros = [1, 2, 3, 4, 5] var palabras = ['Hola', 'mundo', 'cruel'] var deTodito = [true, 'JavaScript', 100, false] console.log(numeros[0]) // 1 console.log(palabras[1]) // 'mundo' console.log(deTodito[2]) // 100 ```
# 🎁 `Object` - Colecciones de `clave: valor` - Modelar la vida real
## Ejemplo ```javascript var persona = { nombre: 'Jose Luis Felix Chilavert', edad: 52, esCrack: true } console.log(persona.nombre) // Jose Luis Felix Chilavert ```
